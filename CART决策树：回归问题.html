CART 决策树回归篇：如何精准预测“具体数值”？

抛开抽象的字母，我们用“卖冰棍”的数据，带你手算一次最小二乘法切分，看懂它如何寻找那个“最完美的边界”。

1. 回归树的核心逻辑：不求“纯度”，但求“稳定”

在之前的分类树中，我们要预测的是类别（去 or 不去），所以我们追求的是“基尼系数”，看分得纯不纯。

但回归树（Regression Tree）要预测的是具体的连续数值（比如：明天能卖出多少根冰棍？房价是多少万？）。此时没有“纯”与“不纯”的概念了，取而代之的是核心指标：平方误差（Squared Error）。

  • 直白的理解： 俗话说“物以类聚，人以群分”。回归树的切分目标是——把数据切成两拨，使得每一拨里的数据都极度相似（方差极小），大家都紧紧围在自己这拨的“平均值”身边。
  • 核心数学公式（残差平方和 SSR）： 假设分到了一个叶子节点里，里面有 $N$ 个数据。我们会算出这 $N$ 个数据的平均值 $\bar{y}$。然后让每个真实数据 $y_i$ 减去平均值，再平方，最后全加起来。公式如下：
$$ SSR = \sum_{i=1}^{N} (y_i - \bar{y})^2 $$

算法目标： 穷举所有的切分一刀，看看哪一刀切下去，左半边加右半边的总 SSR（误差总和）最小！谁最小，谁就是最佳切分点。

2. 极简场景：气温与冰棍销量的预测

为了不被小数和分数搞晕，我为你准备了一组计算结果全为整数的完美数据集。只有 4 天的数据：

编号特征 $X$：最高气温 (℃)标签 $Y$：冰棍销量 (百根)
110℃2
220℃4
330℃12
440℃14

直觉上看，气温在 20℃ 到 30℃ 之间时，销量发生了一个“大跳跃”。我们来看看 CART 算法是不是和我们的直觉一样，能精准找到这个分割线。

3. 穷举试探：CART 的切分算盘

对于连续数值特征，CART 会把相邻特征值取平均数，作为候选的切分点。这里的气温是 10, 20, 30, 40。所以候选切分点有三个：15℃，25℃，35℃。

方案 A：在 15℃ 切一刀（X <= 15）

左边组（编号1）： 销量是 {2}。
左边平均值 $\bar{y}_{左} = 2$。
左边误差 $SSR_{左} = (2 - 2)^2 = 0$

右边组（编号2,3,4）： 销量是 {4, 12, 14}。
右边平均值 $\bar{y}_{右} = \frac{4 + 12 + 14}{3} = \frac{30}{3} = 10$
右边误差 $SSR_{右} = (4 - 10)^2 + (12 - 10)^2 + (14 - 10)^2$
$\quad \quad \quad = (-6)^2 + (2)^2 + (4)^2 = 36 + 4 + 16 = 56$

方案 A 总误差： $0 + 56 = \mathbf{56}$ （误差非常大，说明右边组内部方差太大，很不稳定）

方案 B：在 25℃ 切一刀（X <= 25）

左边组（编号1,2）： 销量是 {2, 4}。
左边平均值 $\bar{y}_{左} = \frac{2 + 4}{2} = 3$
左边误差 $SSR_{左} = (2 - 3)^2 + (4 - 3)^2 = (-1)^2 + (1)^2 = \mathbf{2}$

右边组（编号3,4）： 销量是 {12, 14}。
右边平均值 $\bar{y}_{右} = \frac{12 + 14}{2} = 13$
右边误差 $SSR_{右} = (12 - 13)^2 + (14 - 13)^2 = (-1)^2 + (1)^2 = \mathbf{2}$

方案 B 总误差： $2 + 2 = \mathbf{4}$ （误差极小！大家跟各自平均值贴得非常紧）

方案 C：在 35℃ 切一刀（X <= 35）

和方案 A 对称计算，左边包含 {2, 4, 12}，右边包含 {14}。
左边平均值是 6，误差是 $16 + 4 + 36 = 56$。
右边只有一个点，误差是 0。
方案 C 总误差： $56 + 0 = \mathbf{56}$

数学结论： 方案 B (25℃) 的总误差仅为 4，远远小于方案 A 和 C 的 56。因此 CART 树毫不犹豫地选择了 气温 <= 25℃ 作为根节点的切分标准！

4. 动态演示与数据场景结合 (可视化解析)

通过下方我们特制的动图演示仪，您可以直观地看到“切分线”滑动时，左右两边的组群是如何重新计算平均值（水平实线），以及残差（虚线）是如何变化的。虚线的总长度的平方，就是我们刚刚手算的误差总和。

当前状态：切分点在 15℃，右侧数据参差不齐，总误差高达 56！

5. 最终生成的回归决策树

找到了最佳切分点，这棵树就顺利长出来了。如果有新的数据进来（比如明天天气预报是 22℃），它顺着树枝走，最终会落在左边的叶子上，模型就会预测明天能卖出 3百根 冰棍（即叶子节点内的平均值）。

总结：回归树本质上就是在做“分块平均”。只要算出一刀切下去后的“方差和最小（大家最抱团）”，这一刀就是最佳的选择！
