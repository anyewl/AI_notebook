神经网络动态推导 (二分类：买还是不买)

导师寄语： 在二分类任务中，我们希望输出结果落在 0 到 1 之间，代表“购买概率”。为此，我们使用 Sigmoid 激活函数，将输出层的得分平滑映射为概率。

补充说明：神经网络的三大核心结构

在下方进行操作推导前，让我们先结合动图里的节点，理清神经网络的基本“骨架”：

  • 输入层 (Input Layer - 蓝色节点)： 负责“看”和“听”。它是数据的入口，负责接收外界传入的原始数据（本例中就是具体的“甜度”和“珍珠量”数值）。注意：输入层只负责搬运数据，不进行任何加工或数学计算。
  • 隐藏层 (Hidden Layer - 紫色节点)： 负责“思考”和“总结”。它是输入层与输出层之间的夹层。为什么叫“隐藏”？因为外界看不见里面的运算，它像是网络内部的黑盒。在这里，神经元们通过互连的权重（Weight）和激活函数，疯狂提取并组合出像“高甜度+少珍珠=容易腻”这样深层次的规律。
  • 输出层 (Output Layer - 绿色节点)： 负责“做决定”。它是网络的最后一站，负责将隐藏层提炼好的复杂特征，转化为最终我们要的答案。本例中，它结合计算得出一个 0~1 的“购买概率”。

原理：为什么用 Sigmoid？

输出层 Sigmoid：当我们把奶茶的“口感”和“负罪感”传给输出层后，得到一个原始得分 $z$。Sigmoid 函数 $\sigma(z) = \frac{1}{1+e^{-z}}$ 能将这个得分转化为概率。

  • 如果结果 > 0.5：预测为“买”(1)
  • 如果结果 < 0.5：预测为“不买”(0)

💻 补充：Python 代码实现（与动态推导完全一致）

你可以直接复制以下 Python 代码在本地运行，底层参数与上述前向传播（Forward Propagation）的数学过程和计算结果完全一致：

import numpy as np

# 1. 定义激活函数
def relu(x):
    return np.maximum(0, x)

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

# 2. 初始化网络参数 (与网页动图的边权重完全对应)
# 隐藏层 (输入维度2，输出维度2)
wL1 = np.array([[0.7, -0.3],   # 对应 隐藏层节点1 (接收甜度、珍珠)
                [0.1,  0.8]])  # 对应 隐藏层节点2 (接收甜度、珍珠)
bL1 = np.array([-1.0, -2.0])   # 隐藏层偏置项

# 输出层 (输入维度2，输出维度1)
wL2 = np.array([0.9, -0.6])    # 对应 输出层节点 (接收特征1、特征2)
bL2 = 0.5                      # 输出层偏置项

# 3. 定义测试场景 (3款奶茶的数据)
scenarios = [
    {"name": "1号奶茶", "x": np.array([9, 2])},
    {"name": "2号奶茶", "x": np.array([2, 8])},
    {"name": "3号奶茶", "x": np.array([5, 5])}
]

# 4. 执行前向传播计算
for item in scenarios:
    x = item["x"]
    
    # --- 第一层：隐藏层计算 (Linear + ReLU) ---
    z_hidden = np.dot(wL1, x) + bL1
    a_hidden = relu(z_hidden)
    
    # --- 第二层：输出层计算 (Linear + Sigmoid) ---
    z_out = np.dot(wL2, a_hidden) + bL2
    p_out = sigmoid(z_out)
    
    # 打印结果，保留与页面一致的小数位数
    print(f"--- {item['name']} (甜度:{x[0]}, 珍珠:{x[1]}) ---")
    print(f"隐藏层特征 (a1, a2) : [{a_hidden[0]:.1f}, {a_hidden[1]:.1f}]")
    print(f"输出层得分 (z_out)  : {z_out:.2f}")
    print(f"最终购买概率 (P)    : {p_out:.3f}\n")

面试问题补充：神经网络基础

1. 什么是神经网络 (Neural Network)？

神经网络是一种模拟人脑神经元连接方式的数学模型，它通过多层级的计算来处理复杂的信息。简单来说，它由大量的“神经元”（节点）连接而成，每个连接都有一个“权重”（Weight）。数据输入后，通过逐层加权求和并经过激活函数处理，最终输出结果。其核心能力在于通过反向传播算法（Backpropagation）根据误差自动优化权重，从而学会处理复杂的模式识别、预测等任务。

2. 为什么要使用激活函数 (Activation Function)？

如果没有激活函数，神经网络的每一层本质上都是线性变换（$z=wx+b$）。无论叠加多少层，线性变换的组合依然是线性的。使用激活函数（如 ReLU, Sigmoid, Tanh）引入了非线性因素，使得神经网络能够拟合复杂的非线性函数，从而具备解决实际复杂问题的能力。

3. 什么是“反向传播” (Backpropagation)？

这是训练神经网络的核心算法。它利用链式法则（Chain Rule），从输出层开始，计算预测结果与真实标签之间的误差（Loss），然后将这个误差沿着网络向回传，计算出每一层权重对误差的贡献（梯度）。最后，利用梯度下降法更新权重，让误差变小。

4. 神经网络为什么时候变深？什么时候变宽？ (Backpropagation)？

什么时候需要变宽？ 当你的输入数据本身包含海量且互不相关的低级特征时。你需要足够的“工位”去捕捉这些平行的信息，防止信息在早期网络层就因为维度太低而产生丢失（信息瓶颈）。

什么时候需要变深？ 当你的任务逻辑非常复杂，需要高度抽象的理解时（比如图像识别、自然语言处理）。你需要网络能够一步步抽丝剥茧，将底层的像素/字符组合成高层的语义/逻辑。
